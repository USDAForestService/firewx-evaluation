This folder contains the R script used for extracting the NDFD forecast data from the 2.5 km forecast grids. Six variables are extracted (air temp, rh, wind speed, wind direction, precipitation, cloud cover) from the node located closest to the lat/long of the weather station. R_extract_mypass.R is the script for the 2015 forecast for the Jan-June forecast data located on the 'my passport' external harddrive. R_extract_elem.R is the script for 2015 forecast data for July-Dec, located on the 'elements' external harddrive.
